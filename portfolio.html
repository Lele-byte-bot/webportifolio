NetWatch – Network Traffic Analyzer & Threat Monitor

NetWatch is a mobile cybersecurity app built with React Native that helps users visualize and understand network traffic patterns. Designed for education and awareness, it allows users to import or simulate traffic logs, view interactive charts of IPs, ports, and protocols, and receive alerts when suspicious activity is detected. With color‑coded severity levels and clear explanations of common attack patterns like port scans or brute force attempts, NetWatch empowers learners, IT students, and professionals in Zimbabwe and beyond to recognize threats and strengthen digital defenses.

SecureLearn – Web App Security Training

SecureLearn is an interactive mobile cybersecurity training app built with React Native that teaches web application security through hands-on learning. Users can explore common vulnerabilities like SQL injection, XSS, and CSRF through simulated scenarios, with step-by-step guidance and real-time feedback. The app includes OWASP Top 10 coverage, progress tracking, and certificate generation upon completion. Designed for developers, security professionals, and students, SecureLearn makes complex security concepts accessible and engaging through gamified learning experiences.

QinisoPay – Truth in Payments

QinisoPay is an innovative mobile cybersecurity awareness app built with React Native that focuses on protecting users from mobile money fraud in Zimbabwe and across Africa. Through interactive simulations and real-world scenarios, users learn to identify common scams like fake payment requests, phishing attempts, and social engineering tactics. The app features gamified learning modules, progress tracking, and practical tips for secure mobile transactions. With its user-friendly interface and culturally relevant content, QinisoPay empowers everyday users to make informed decisions and stay safe in the digital economy.

Web development React Native Interactive Simulations

CyberGuard – Digital Defense & Awareness App

CyberGuard is a comprehensive mobile cybersecurity toolkit built with React Native that provides users with essential digital defense tools and awareness resources. The app features a built-in password strength checker, secure password generator, phishing detection tools, and educational content about common cyber threats. Users can scan URLs for malicious content, learn about safe browsing practices, and access a digital security checklist. With its intuitive interface and offline capabilities, CyberGuard serves as both a practical security tool and an educational resource for users looking to enhance their digital literacy and protect themselves online.
